## 분석 개요
분석을 수행하기 위해 설계된 분석 문제는 다음과 같다.

문제 1. 학생들의 LMS 만족도 조사 결과는 어떠한가?
문제 2. 문/이과 학생들의 LMS 만족도 조사 결과에는 차이가 존재하는가?
문제 3. 학부/대학원생들의 LMS 만족도 조사 결과에는 차이가 존재하는가?

<br/>
